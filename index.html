Before getting started with your application, play our loop game. It's fun


To play the loop game, create of number sequence in your mind. Enter the first number of that sequence into the First Number field below. Enter the last number of the sequence into the Last Number field, then click the PLAY! button.

You will see an amazing result. Our magical loop engine will create a multiplication table generated from the input numbers, continuously until the result of the multiplication reaches the last number in your sequence. If the last number is a multiple of the first number, the highest number in the multiplication table will be the last number you entered, otherwise, it will be next lower multiple number in the sequence.
